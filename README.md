# Rust STL Viewer

STL viewer written in Rust.

## Installation

```bash
git clone https://github.com/mantreshkhurana/rust-stl-viewer.git
cd rust-stl-viewer
cargo install --path .
```

## Usage

```bash
stl-viewer path/to/model.stl
```

## Test

```bash
cargo run models/model-1.stl 
```

> Example: `rust-stl-viewer models/model-1.stl`
